Patch
=====
TestsLogConfigurationHelper.java innehåller en bug i version 3.7.0 av Mule som påverkar laddning av log4j2-test.xml i Windows.

MuleSoft har gjort en rättning som löser detta och patchen har hämtats från mule i GitHub.
MULE-9027: Fixed loading of the Log4j config file, making it compatible with Windows
